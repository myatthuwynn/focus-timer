## How to Use

1. **Set your durations**
   - Adjust "Work" duration (default: 25 minutes)
   - Adjust "Break" duration (default: 5 minutes)
   - Recommendations:
     - Light tasks: 25min work / 5min break (classic Pomodoro)
     - Clinical documentation: 45min work / 10min break
     - Deep analysis: 60min work / 15min break

2. **Start your session**
   - Click "Start" to begin the work timer
   - Status shows: "Focus Work" (green)
   - Progress bar fills as time elapses

3. **Session switches automatically**
   - When work timer ends, break timer starts automatically
   - Audio notification alerts you to the transition
   - Status changes to "Break Time" (blue)
   - Sessions completed counter increments

4. **Pause or reset anytime**
   - Click "Pause" to temporarily stop the timer
   - Click "Reset" to clear and start fresh
   - Pause preserves your session count

5. **Track your progress**
   - View "Sessions completed" for motivation
   - Monitor "Total focus time" accumulated
   - Use this data to optimize your work blocks

## Tips for Maximum Effectiveness

1. **Set realistic durations** - Start with 25min if new to focused work
2. **Use breaks intentionally** - Move, hydrate, stretch (not social media)
3. **Remove notifications** - Silence phone/Slack during work blocks
4. **Track patterns** - Notice which durations suit different tasks
5. **Build habits** - Consistency over intensity
6. **Combine with intention** - Know exactly what you'll accomplish before starting
